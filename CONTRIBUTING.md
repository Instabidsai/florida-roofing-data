# Contributing to Florida Roofing Data

This dataset is maintained by [Goliath Roofing of Florida](https://goliathroofingofflorida.com).

## Data Sources

- **Cost data**: Based on 1,200+ completed roofing projects across Miami-Dade, Broward, and Palm Beach counties
- **Permit data**: Sourced from county building department public records
- **Insurance data**: Florida Office of Insurance Regulation public reports
- **Hurricane data**: NOAA Storm Events Database
- **Building code data**: Florida Building Commission official publications
