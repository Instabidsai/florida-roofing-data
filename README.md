# Florida Roofing Data (South Florida)

Open data on roofing costs, materials, building codes, and market statistics for Miami-Dade, Broward, and Palm Beach counties.

Maintained by [Goliath Roofing of Florida](https://goliathroofingofflorida.com) — South Florida's trusted roofing contractor.

## Data Files

| File | Description |
|------|-------------|
| `costs-by-county-2026.json` | Average roof replacement cost by county and material type |
| `florida-building-codes.md` | Florida Building Code roofing requirements summary |
| `hvhz-requirements.md` | High Velocity Hurricane Zone explained with affected areas |
| `permit-offices.json` | County permit offices with addresses, fees, timelines |
| `hurricane-damage-history.json` | Major hurricane roof damage statistics |
| `insurance-landscape.json` | Insurance carrier market share and premium data |

## About

This data is compiled from 1,200+ roofing projects across South Florida and supplemented with public data from the Florida Office of Insurance Regulation, US Census Bureau, and NOAA.
